# 이용약관

Muse는 기술 서비스를 제공하며, Muse 사용의 적법성에 대한 모든 책임은 사용자에게 있습니다.
사용자 혹은 외부인이 Muse를 통해 컨텐츠를 다운로드하는 것과 관련해 Muse 측은 어떠한 법적 책임도 지지 않습니다
